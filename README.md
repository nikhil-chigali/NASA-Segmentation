## Software dependencies

## Setup

1. Create a Python `venv` environment

    ```sh
    python3 -m venv .venv
    ```

2. Activate the environment

    ```sh
    .\.venv\Scripts\activate
    ```

3. Install requirements

    ```sh
    pip install -r requirements.txt
    ```
